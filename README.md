# Mistri Interior Attendance System

A full-stack HR management system built for Mistri Interior to handle employee attendance, photo verification, and location tracking.

## Features
* **Smart Punch In/Out:** Location and selfie-based attendance tracking.
* **Role-Based Dashboards:** Separate UIs for Admin and Employees.
* **Live History:** Interactive calendar with color-coded status (Present, Absent, Forgot Out, Forced Out).
* **Location Verification:** Direct integration with Google Maps to verify punch-in locations.
* **Admin Controls:** View daily logs, force employee logouts, and download monthly PDF reports.
* **Security:** OTP-based "Forgot Password" system via email/console.

## Tech Stack
* **Frontend:** Flutter (Android & iOS ready)
* **Backend:** Django & Django REST Framework
* **Database:** SQLite (Development)
